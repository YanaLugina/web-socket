# About
Testing WebSocket. Creating a live-connect chat in NodeJS.

## Need:

+ Nodejs and npm
+ http-server
+ ws
+ browser
+ text editor

## for Run

1. Clone project
2. Install ws
3. Install http-server
4. Run server ``` node serever```
5. Run client ``` http-server```
6. Go to browser: http://localhost:8080/
